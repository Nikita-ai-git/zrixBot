How to Integrate the Chatbot

Adding our chatbot to your website is incredibly simple. Just follow these steps:

Step 1: Basic Integration (Local File)

Add this code before your closing </body> tag:

<script src="chatbot.js"></script>

Step 1: CDN Integration (Recommended)

For optimal performance, use CDN hosting:

<!-- CloudFlare CDN -->
<script src="https://cdn.yoursite.com/chatbot.js"></script>

<!-- AWS CloudFront -->
<script src="https://d1234567890.cloudfront.net/chatbot.js"></script>

<!-- jsDelivr (GitHub-based CDN) -->
<script src="https://cdn.jsdelivr.net/gh/username/repo@main/chatbot.js"></script>

<!-- unpkg CDN -->
<script src="https://unpkg.com/your-package@latest/chatbot.js"></script>

Step 2: Optional Customization

Customize the chatbot by adding configuration before the script:

<script> window.ChatbotConfig = { primaryColor: '#667eea', secondaryColor: '#764ba2', title: 'Customer Support', avatarText: 'CS', greeting: 'Hi! How can we help you today?', position: 'bottom-right', autoOpen: false, showNotification: true }; </script> <script src="https://cdn.yoursite.com/chatbot.js"></script>

CDN Setup Guide

Choose your preferred CDN service:

# CloudFlare Setup
1. Sign up at cloudflare.com
2. Add your domain
3. Upload chatbot.js to your origin server
4. Use: https://cdn.yoursite.com/chatbot.js

# AWS CloudFront Setup
1. Create S3 bucket with chatbot.js
2. Create CloudFront distribution
3. Use: https://d1234567890.cloudfront.net/chatbot.js

# jsDelivr (Free GitHub CDN)
1. Upload to GitHub repository
2. Use: https://cdn.jsdelivr.net/gh/user/repo@main/chatbot.js

Cross-Origin Setup (CORS)

For remote servers, ensure CORS headers are set:

# Apache (.htaccess)
Header set Access-Control-Allow-Origin "*"
Header set Access-Control-Allow-Methods "GET, OPTIONS"

# Nginx
add_header Access-Control-Allow-Origin *;
add_header Access-Control-Allow-Methods 'GET, OPTIONS';

# Node.js/Express
app.use((req, res, next) => {
  res.header('Access-Control-Allow-Origin', '*');
  next();
});
